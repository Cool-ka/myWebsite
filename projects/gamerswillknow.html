GAMERS WILL KNOW

Gamers Will Know is a blog website for industry-specific stories and analyses, strictly connected with video games industry. It is based on Wordpress CMS, with the theme being built up entirely from scratch.

The work involved designing and coding the blog theme, preparing all the supporting graphic creatives (logo, favicon, images for first posts), setting up access to CMS for the client, building up a custom ‘About me’ page, setting up Google Analytics profile, and ongoing support – in case the client has any questions regarding Wordpress CMS or website structure.
