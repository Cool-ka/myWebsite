SEE GAME

TAP THEM ALL

This mobile game project for Android mobile phones required designing and creating UX elements and art. The work involved creating a menu, in-game interface design, textures for game elements and backgrounds.

Furthermore, in order to meet all the requirements connected with publishing a game in the Google Play store, the project also involved creation of marketing assets – nicely looking screenshots, game icons and logo. The style of both marketing assets and in-game art had to be consistent, so to avoid confusing the end user.
