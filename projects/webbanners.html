WEB BANNERS

Supporting advertising materials, made for online promotion of games, movies and hardware. Considering the nature of asset production for marketing purposes, these projects usually involved creating ads in various sizes and formats, adhering to marketing and technical guidelines, ensuring that all the elements requested by clients are present and clearly visible in all the creatives.

The assets created included smaller forms (such as billboards or rectangles), as well as larger ones (interstitials, site takeovers).

What is more, I have also worked with adserving tools, where I set up not only the abovementioned creatives, but also rich media ads. These involved adjusting the code behind ads, so they are both functional and trackable.
